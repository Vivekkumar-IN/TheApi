SaavnAPI¶

class TheApi.SaavnAPI¶

A class for interacting with the Saavn API.

This class provides methods for searching songs, albums, artists, and playlists globally on the Saavn platform.

from TheApi import SaavnAPI

api = SaavnAPI()
await api.search("Fuck Millionaire")
