search

async SaavnAPI.search(query: str)

Searches globally for songs, albums, artists, and playlists.

Parameters:

query (str) – The search query. This can be a song name, album name, artist name, or playlist name.

Returns:

dict –

The search results containing information about the songs,

albums, artists, and playlists matching the query.

Example

response = await api.search(“Despacito”) print(response)
