github_search¶

async Client.github_search(query, search_type='repositories', max_results=3)¶

Searches GitHub for various types of content.

Parameters:

  • query (str) – The search query.

  • search_type (str, optional) – The type of search. Can be one of: - “repositories” - “users” - “organizations” - “issues” - “pull_requests” - “commits” - “topics”

    Defaults to “repositories”.

  • max_results (int, optional) – The maximum number of results to return. Defaults to 3.

Returns:

list – A list of search results or an error message.
